\doxysubsection{Detailed Description}
Manages a queue of requests and distributes them among active web servers. 


\begin{DoxyItemize}
\item The \mbox{\hyperlink{classLoadBalancer}{Load\+Balancer}} dynamically scales the number of active servers based on queue size, blocks malicious IPs, and tracks statistics over a specified run time. 
\end{DoxyItemize}

{\footnotesize\ttfamily void Load\+Balancer\+::run (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}



The main simulation loop that manages servers, routing, and dynamic scaling. 

Executes the main simulation loop.

The simulation runs for the specified duration, cycling through\+:
\begin{DoxyItemize}
\item Time advancement and request completion detection
\item Server assignment of queued requests
\item Random injection of new requests ($\sim$10\% per cycle)
\item Dynamic server scaling (every 30 cycles)\+:
\begin{DoxyItemize}
\item Add server if queue $>$ 80$\ast$server\+\_\+count
\item Remove server if queue $<$ 50$\ast$server\+\_\+count and server is idle
\end{DoxyItemize}
\end{DoxyItemize}

All activity is logged to the log file and displayed with color output.
